# Python Algorithms Collection

This repository contains implementations of important **Data Structures and Algorithms (DSA)** in Python. It is designed for learning, practice, and understanding commonly used algorithms in computer science.

## Files Included

### `dijkstra.py`

Implements **Dijkstra’s Algorithm** to find the shortest path from a source node to all other nodes in a weighted graph.

### `interpolation_search.py`

Implements **Interpolation Search**, an efficient searching algorithm for uniformly distributed sorted arrays.

### `kruskal-prims.py`

Contains implementations of **Kruskal’s Algorithm** and **Prim’s Algorithm** for finding the **Minimum Spanning Tree (MST)** of a graph.

### `naive-rabin-kmp.py`

Includes implementations of three string matching algorithms:

* **Naive String Matching**
* **Rabin-Karp Algorithm**
* **Knuth-Morris-Pratt (KMP) Algorithm**

## How to Run

1. Make sure Python is installed on your system.
2. Open a terminal in the project folder.
3. Run any file using:

```bash
python filename.py
```

Example:

```bash
python dijkstra.py
```
